Acerca de este Proyecto

Este proyecto es un sistema sencillo para manejar productos, clientes e inventarios usando Flask y MySQL.

Qué puedes hacer con este proyecto:

  • Ver y administrar productos disponibles.
  • Registrar nuevos clientes y consultar los existentes.
  • Llevar control de los inventarios de productos.
  • Agregar productos nuevos desde la web fácilmente.

Tecnologías que usé:

  • Python con Flask para la parte web.
  • MySQL (XAMPP) para guardar los datos.
  • HTML y CSS para las páginas y el diseño.

Este proyecto ayuda a practicar la conexión entre Flask y bases de datos, y a construir páginas web que muestran información dinámica.
